# Search News by Country 

A simple and responsive **News Search Web Application** that allows users to fetch the latest news based on a **country code or keyword**. The application is built using **HTML, CSS, and JavaScript** and consumes real-time data from the **NewsData.io API**.

---

## Features

* Search latest news by **country code** (e.g., `in`, `us`, `uk`)
* Search news using **keywords** (e.g., technology, business, sports)
* Displays:

  * News headline
  * Short description
  * Direct link to full article
* Clean and responsive UI
* Deployed using **GitHub Pages**

---

## Tech Stack

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive layout
* **JavaScript (ES6)** – API integration and DOM manipulation
* **NewsData.io API** – Fetching real-time news data

---

## How It Works

1. User enters a country code or keyword in the search box.
2. The app sends a request to the NewsData.io API.
3. Latest news articles are fetched and displayed dynamically.
4. Each article includes a "Read more" link for full details.
